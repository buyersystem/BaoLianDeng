Features

Everything you need for proxy management on iOS

🔗

Subscription Management

Add, edit, refresh, and switch between proxy subscriptions. Supports Clash YAML and base64 formats.

🌐

Proxy Node Selection

Browse nodes by subscription with protocol icons and latency indicators for optimal routing.

📊

Traffic Analytics

Daily bar charts, session stats, and monthly summaries for proxy-only traffic monitoring.

📝

Config Editor

In-app YAML editor with validation for both local config and subscription configs.

🔄

Proxy Groups

View and switch proxy groups via Mihomo's REST API. Supports rule, global, and direct modes.

📋

Tunnel Logs

Real-time log viewer for debugging the network extension with configurable log levels.

Installation

Sideload BaoLianDeng onto your iPhone

⭐ AltStore / SideStore

Add this source URL to receive automatic updates:

Or install manually from a downloaded IPA via AltStore or SideStore.

📥 Sideloadly

  1. Download Sideloadly (macOS / Windows)
  2. Download BaoLianDeng.ipa
  3. Connect your iPhone via USB
  4. Drag the IPA into Sideloadly and sign with your Apple ID
  5. Trust the profile: Settings → General → VPN & Device Management

🔧 Manual Re-sign

For advanced users: re-sign the IPA with codesign on macOS. Both the app and the network extension require App Groups and Packet Tunnel entitlements.
